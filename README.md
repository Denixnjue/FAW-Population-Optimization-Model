# FAW Stage-Structured Modeling & Optimal Control

## Executive Summary
This project presents a **Temperature-Dependent Stage-Structured Mathematical Model** for the Fall Armyworm (FAW). It moves beyond basic growth models by accounting for thermal thresholds that govern the transition from Egg to Larva to Adult.

## Advanced Features
- **Numerical Analysis:** Solved a system of non-linear Ordinary Differential Equations (ODEs) using the Runge-Kutta 4th Order method.
- **Optimal Control Theory:** Integrated a control variable ($u$) to simulate targeted pesticide application, identifying the 10-day window that minimizes crop damage.
- **Sensitivity Analysis:** The model demonstrates that Larval peak intensity is highly sensitive to ambient temperature fluctuations, requiring adaptive intervention strategies.

## Impact
This tool acts as a **Decision Support System**, helping agricultural officers determine the most cost-effective moment to intervene in a pest outbreak.
